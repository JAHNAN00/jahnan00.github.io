## 本文说明

用于整理 C++ 电话面和实习面里最常见的基础八股题，只保留最核心结论。

## 指针和引用的区别

**指针是存地址的变量，引用是对象的别名。**

* 指针可以为 `nullptr`，引用定义时必须绑定对象。
* 指针可以改指向，引用一旦绑定后不能改绑。
* 指针访问要 `*` 或 `->`，引用用起来和原对象一样。

## 堆和栈的区别

**栈由系统自动管理，堆由程序主动申请和释放。**

* 栈上的变量随作用域结束自动销毁，分配回收快。
* 堆上的对象生命周期更灵活，但容易内存泄漏。
* 栈空间通常较小，堆空间通常较大。

## `new` 和 `malloc` 的区别

**`malloc` 只分配内存，`new` 不仅分配内存，还会调用构造函数。**

* `malloc/free` 是 C 的库函数，`new/delete` 是 C++ 运算符。
* `malloc` 不会初始化对象，`free` 不会调用析构函数。
* `new` 失败默认抛异常，`malloc` 失败返回 `NULL`。

## 构造函数能不能是虚函数

**构造函数不能是虚函数，因为对象还没构造完成时，没法基于完整多态去分发。**

* 虚函数调用依赖虚表。
* 构造函数执行时，对象本身还在构造过程中。
* 析构函数常设为虚函数，是为了通过基类指针删除派生类对象时能完整析构。
