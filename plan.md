# 重构计划（VuePress -> Vue + Vite）

## 目标

将当前博客站重构为综合个人门户站，包含：

1. 首页（个人介绍）
2. 作品（卡片流 + 详情页）
3. 博客（列表 + 标签/分类筛选 + 详情页）

并保持现有 GitHub Actions 工作流不变（push `docs` 分支后构建并发布到 `main` 分支）。

## 约束

- 技术栈：Vue + Vite。
- 不兼容旧链接（`/blogs/*`、`/posts`、`/tags/*` 不做重定向）。
- 作品数据先采用 JSON。
- 博客内容迁移到 `src/content/blog`。
- 保留深浅色切换，整体风格简约。

## 执行步骤

1. 建立 Vite 工程入口与基础脚本。
2. 将构建输出目录固定到 `docs/.vuepress/dist` 以兼容现有 CI。
3. 建立路由与三大页面骨架：首页、作品、博客。
4. 迁移博客 Markdown 到 `src/content/blog`，实现 front matter 解析、列表、筛选、详情渲染。
5. 建立作品 JSON 数据与卡片/详情页展示。
6. 建立全局主题变量与深浅色切换（跟随系统 + 手动切换持久化）。
7. 安装依赖、构建验证、记录实验过程。
